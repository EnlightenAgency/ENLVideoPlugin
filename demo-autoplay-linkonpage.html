8 Video will autoplay in a LightBox when this page is loaded

This video automatically plays in a LightBox when the page is loaded. The link to the video is present in the HTML, keeping it accessible to the user after the LightBox has been closed. The URL to autoplay the video must include a query string containing a path to the video that matches the path in the video link on the page. This ensures that only that element is targeted.

You will need an additional file called ENLvideo.autoplay.js that should be included after the ENLvideo.js plugin file. You also must include the ColorBox files. You will need to make a call to the plugin to create the video in a LightBox, to match the call made in the autoplay JS. This ensures that the video will still be able to play properly and as expected when the page is accessed without the query string that initiates the autoplay.

URL with Query String

http://[path]?videoURL=video/FE_768x432_h264_750Kbps.mp4

HTML

<a href="video/FE_Serum_768x432_h264_750Kbps.mp4" class="videoLink" rel="videoparams::576|324|300"> This is the link to the video that auto-played. Clicking it plays the video again. </a>

Include jQuery files

<script src="js/jquery.ENLvideo.min.js"></script>
<script src="js/jquery.ENLvideo.autoplay.min.js"></script>

jQuery

$('a.videoLink').ENLvideo({ lightBox: true,
hideVideoLink: false });
