7 Video will autoplay in a LightBox when the page is loaded

The video automatically plays in a LightBox when the page is loaded. There is no HTML or link for the video in the page DOM. This is done by including the ENLvideo.autoplay.js file after the plugin and adding a query string to the URL with the video location, width, height, and duration. When doing this, you do not need to include a link to the video on the page itself (though you can if desired, as shown in the Auto-play Demo: Link on Page example).

You will need an additional file called ENLvideo.autoplay.js that should be included after the ENLvideo.js plugin file. You also must include the ColorBox files. You do NOT need to include a call to the plugin in this case. The autoplay JS constructs the proper elements and makes the call for you.

Note: If the video did not play, click here to reload the page with the necessary autoplay parameters.

URL with Query String

http://[path]?videoURL=video/FE_768x432_h264_750Kbps.mp4&videoWidth=768&videoHeight=432&videoDuration=114

Include jQuery files

<script src="js/jquery.ENLvideo.min.js"></script>
<script src="js/jquery.ENLvideo.autoplay.min.js"></script>
